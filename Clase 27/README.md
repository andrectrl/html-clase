# Calculadora de 3 Números

Página sencilla que permite calcular con tres números: suma, promedio, producto, máximo y mínimo.

Cómo usar
- Abre index.html en tu navegador.
- Introduce tres números y selecciona la operación.
- Haz clic en **Calcular**.

Archivos
- index.html: interfaz y lógica JavaScript.
- styles.css: estilos profesionales y responsive.

Notas
- Funciona offline en cualquier navegador moderno.
